# Simulação da Liga Portuguesa

Isto é uma simulação da liga portuguesa feito em Javascript.

Eu sei que não é mto realista mas não importa

## Coisas que podes fazer

- podes simular jogos
- podes ver a evolução dos clubes
- podes mudar o código 

## Como é que raio posso abrir esta "programa"????

1: Carregue no Código (ou Code se está em inglês) e carregue no Download ZIP

2: Extrai o Ficheiro zip

3: Instalas node.js[https://nodejs.org/en]

4: abres o Prompt de Comando e escreves isto: `cd [insere a localização onde extraistes] && node index.js`
